HTML and CSS course at the University of Scranton

Course description

This is a technical course for students who do not necessarily have a technical background. May not be used as part of any major in the Computing Sciences department. A course that covers fundamental aspects of web design and development.

  • Coding HTML and cascading style sheets.
  • JavaScript to create dynamic web pages.
  • Server-side vs. client-side technologies.
  • Using a Content Management System that separates design from content while making it easy for non-technical users to update a web site.

Interest Aspects

HTML5

HTML5 (HyperText Markup Language, versi√≥n 5) It is the fifth revision of the HTML standard. Its core aims have been to improve the language with support for the latest multimedia while keeping it easily understood by computers and devices.

CSS3

CSS3 is the latest evolution of the Cascading Style Sheets language and aims at extending CSS2.1. It brings a lot of long-awaited novelties, like rounded corners, shadows, gradients , transitions or animations , as well as new layouts like multi-columns , flexible box or grid layouts.

Javascript

JavaScript (JS) is a dynamic computer programming language. It is most commonly used as part of web browsers, whose implementations allow client-side scripts to interact with the user, control the browser, communicate asynchronously, and alter the document content that is displayed.

Bootstrap

Bootstrap is a free collection of tools for creating websites and web applications. It contains HTML and CSS-based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions.

JQuery

jQuery is a JavaScript library that allows web developers to add extra functionality to their websites. It is open source and provided for free under the MIT license. In recent years, jQuery has become the most popular JavaScript library used in web development.

PHP & Node.js

PHP is free software released under the PHP License. PHP can be deployed on most web servers and also as a standalone shell on almost every operating system and platform, free of charge.

Node.js is a software platform for scalable server-side and networking applications. Node.js applications are written in JavaScript, and can be run within the Node.js runtime on Mac OS X, Windows and Linux with no changes.
